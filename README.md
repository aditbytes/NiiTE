# NITE

**Nightlife. Verified. Playable.**

A premium landing page for NITE — a real-time, verified nightlife intelligence layer presented as a playable metaverse-style city map.

![Hero Screenshot](hero-bg.png)

---

## Overview

NITE transforms nightlife discovery by providing real-time, verified data about venues, crowd dynamics, and city-wide nightlife patterns. This repository contains the landing page showcasing the platform's vision and capabilities.

## Features

- **Three.js City Visualization** — Interactive 3D city grid with pulsing venue nodes
- **GSAP Scroll Animations** — Smooth, cinematic scroll-triggered effects
- **Real-time Dashboard Demo** — Live-updating venue analytics mockup
- **Responsive Design** — Mobile-first approach with elegant breakpoints
- **Dark Mode Aesthetic** — Premium futuristic visual language

- **HTML5** — Semantic structure

- **GSAP + ScrollTrigger** — Animations (via CDN)

## Quick Start

```bash
# Clone the repo
git clone https://github.com/yourusername/nite.git
cd nite

# Open in browser
open index.html

# Or serve locally (recommended)
npx serve .
```
